## Requirements
- iOS 17.0 +
- MacOS Catlina or updated one

## Setup
 1) Fork this project
 2) Clone the forked project to your machine
 3) Open the `Upstox.xcodeproj` file in Xcode
 4) `cmd` + `R` to run your favourite app.
